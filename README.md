## Installation

Create and activate a virtual environment:

```bash
python -m venv .venv
source .venv/bin/activate
```

Install the required packages:

```bash
pip install torch torchvision torchaudio jupyter matplotlib
```

Start Jupyter Notebook:

```bash
jupyter notebook
```
